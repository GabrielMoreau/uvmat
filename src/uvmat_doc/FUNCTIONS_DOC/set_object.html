PURPOSE

'set_object': GUI to edit a projection object

SYNOPSIS

DESCRIPTION 

'set_object': GUI to edit a projection object
------------------------------------------------------------------------

 OUTPUT:
 hset_object: handle of the GUI figure
 
 INTPUT:
 data: structure describing the object properties
